Interface
Minimal Mode (Headless)
Hide tooltip button and popup UI. Use hotkeys only. Perfect for power users.
Show "What's this?" button
Show yellow button when text is selected on a page.
Backend
Worker / API base URL

Point the extension at your deployed worker. Use an HTTPS base URL such as https://your-worker.workers.dev.

Save requests permission for that origin and stores the URL locally. Follow the setup steps in the project README before using the extension.

Hotkeys
Smart Explain selected text Ctrl+Shift+E
Smart Analyze full page Ctrl+Shift+R
Copy last result Ctrl+Shift+C

To change hotkeys go to chrome://extensions/shortcuts
